### R Basics

You can perform following arithmetic operations in R

- Addition: `+`
- Subtraction: `-`
- Multiplication: `*`
- Division: `/`
- Exponentiation: `^`
- Modulo: `%%`

Run the following chunk to see results:

```r=
5 + 5 

5 - 5 

3 * 5

(5 + 5)/2 

# Exponentiation
2^5

# Modulo
28 %% 8
```
